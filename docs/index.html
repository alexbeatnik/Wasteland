WASTELAND TERMINAL v0.2

A highly secure, offline-first LLM inference client built in pure C with a vintage PC-inspired amber-on-black CRT terminal aesthetic. It runs entirely offline once a model is loaded, using Linux seccomp to physically prevent the process from opening any new sockets for the rest of its lifetime.

--- CORE PROTOCOLS ---

Smart Reasoning Automatic detection of <think> blocks. The model's thought process is rendered in a separate dimmed box, keeping the final output clean.
Context Compaction Drop the oldest turns, save the compacted history to disk, and mirror the result into the inference thread without reloading the model.
Agent Mode (ReAct) Built-in tool-using ReAct agent loop with sandboxed file system access (read_file, list_dir, write_file, apply_edit).
Multi-Turn Memory Create, load, and switch between named chat sessions. Persisted locally with RC4 obfuscation.

--- SYSTEM SPECS ---

  • Core Engine: Pure C11 via llama.h
  • Interface: Nuklear Immediate-Mode GUI (SDL2/OpenGL 2.1)
  • Security: Linux seccomp-bpf (SCMP_ACT_KILL_PROCESS on new sockets)
  • Networking: libcurl (download path only; strictly isolated)
